#확률변수의 기대값과 분산
 1. 기대값
  E(x) = ∑xP(x)
  Ex) 주사위를 던질때 기대값은
   1*1/6 + 2*1/6 + 3*1/6 + 4*1/6 + 5*1/6 + 6*1/6 = 3.5
  *기대값의 성질
   1) E(a) = a
   2) E(aX) = aE(X)
   3) E(aX土b) = aE(X)土b
   4) E(aX土bY) = aE(X)土bE(Y)
   5) E(XY) = E(X)E(Y) (X와 Y는 확률적으로 독립)
 2. 분산
  *분산의 성질
   1) Var(a) = 0
   2) Var(aX) = a^2Var(X)
   3) Var(X+Y) = Var(X) + Var(Y)+2Cor(X,Y) //Cor은 공분산
      Var(X+Y) = Var(X) + Var(Y), X와 Y는 확률적으로 독립
   4) Var(X-Y) = Var(X) + Var(Y) - 2Cor(X,Y)
      Var(X-Y) = Var(X) + Var(Y), X와 Y는 확률적으로 독립

#확률분포
 1.확률분포 : 미래에 발생할 사건에 대한 확률을 나열한 것
  *표준정규분포 : 종모양
 2. 이항분포 : 연속적인 베르누이 시행을 거쳐 나타나는 확률분포
  *베르누이 시행
  *베르누이 분포

이항분포의 평균(μ)과 분산(σ^2)은 
μ = np...

이항분포의 확률은 n번의 시행에서 성공확률(p)이 r번 나타날 확률이며
n번의 시행에서 r번 관찰되는 것은 조합 nCr로 표현할 수 있다.

r번 성공할 확률과 (n-r)번의 실패할 확률을 곱하면
이항분포의 확률함수는

P(X=r)=(n!/r!*(n-r)!) * p^r * (1-p)^(n-r)


 3.포아송분포 : 특정한 사건이 발생할 가능성이 매우 드문 경우의 확률분포
  ex) 책에 오타가 얼마나 있는지 등
  P(X=x) = nCx * p^x * (1-p)^(n-x)
